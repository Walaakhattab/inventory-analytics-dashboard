## Project Features

- Built an interactive Power BI dashboard for inventory and supply chain analysis.
- Cleaned and transformed data using Power Query.
- Created DAX measures for inventory and supply chain KPIs.
- Designed an executive dashboard with interactive slicers.
- Implemented data modeling and relationships between tables.
- Used charts, KPI cards, gauge, line, bar, and donut visuals to present business insights.

- ## KPIs

- Warehouse Utilization
- Days Sales of Inventory (DSI)
- Inventory Turnover Ratio
- Lead Time
- Transportation Cost
- Inventory Level
- Units Sold
- Backorder Count
  
## Tools Used

- Power BI
- Power Query
- DAX
- Data Modeling
- Microsoft Excel
